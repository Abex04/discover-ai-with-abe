# ✨ Discover AI with Abe

The most curated directory of 125+ AI tools — with honest pricing, real descriptions, and Abe's personal picks.

## 🚀 Features
- 125+ AI tools across 10 categories
- Live search + category filtering
- My Picks — save & review your favorite tools
- Ask Abe — AI-powered recommendations
- Contact page with Formspree email + WhatsApp
- Dark / Light mode
- Fully mobile responsive

## 🛠️ Tech Stack
- React 18 + Vite
- Formspree (contact form emails)
- Claude API (Ask Abe AI chat)
- Deployed on Vercel

## 📦 Setup

```bash
# 1. Clone the repo
git clone https://github.com/Abex04/discover-ai-with-abe.git
cd discover-ai-with-abe

# 2. Install dependencies
npm install

# 3. Run locally
npm run dev

# 4. Build for production
npm run build
```
